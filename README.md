### Prerequisites

* [Anaconda](https://conda.io/miniconda.html) - Python Package Manager
* [Android Studio](https://developer.android.com/studio/install) - Android Development Environment
* [Arduino](https://www.arduino.cc/) - General Purpose Microcontroller
* [Atom](https://www.atom.io/) - Text Editor
* [Audacity](https://www.audacityteam.org/) - Audio Editor
* [Autodesk Fusion 360](https://www.autodesk.com/products/fusion-360/overview) - 3D CAD
* [Blender](https://www.blender.org/) - 3D CAD
* [DokuWiki](https://www.dokuwiki.org/) - Wikipedia
* [Git](https://git-scm.com/) - Version Control
* [GitHub Desktop](https://desktop.github.com/) - Version Control GUI
* [GitLab](https://gitlab.com/) - Version Control
* [InkScape](https://inkscape.org/en/) - Vector Graphics
* [jQuery](http://jquery.com/) - JavaScript Library
* [KiCad PCB](http://kicad-pcb.org/) - Printed Circuit Boards
* [LMMS](https://lmms.io/) - Digital Audio Workstation
* [Python](https://www.python.org/) - General Purpose Programming Language
* [Raspberry Pi](http://raspberrypi.org/) - General Purpose PC
* [React](https://reactjs.org/) and [React Native](http://reactnative.com/) - Frameworks for building user interfaces
* [Read The Docs](https://readthedocs.org/) - Documentation
* [Unity3D](https://unity3d.com/) - Game Development Platform
* [Wordpress](https://wordpress.org/) - Website Creation
